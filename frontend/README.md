# OCR Frontend Application

React + Vite + Material-UI application for image OCR processing.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Environment Variables

Create a `.env` file in the root directory:

```
VITE_API_BASE_URL=http://localhost:5000
```

For production, update the URL to your EC2 backend instance.

## Technologies

- React 18
- Vite
- Material-UI (MUI)
- Axios
- React Dropzone
